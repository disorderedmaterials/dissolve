## Import

### File Types

|Keyword|Description|
|:---:|-----------|
|`xyz`|Appended XMol-style xyz coordinates. Line 1 contains the number of atoms N. Line 2 contains a title string. The next N lines contain "element  rx  ry  rz". This format is repeated for each frame.|
